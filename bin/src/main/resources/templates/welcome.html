Welcome at Car Dealer Application!

This is a Demo Car Dealer Shop Application based on Spring-Boot and In-Memory HSQL Database!
(MySQL configuration is also available, please check it in the Source Code)

Features:
  • You can Create an Advertisement as a Salesman or Buy a Car as a Customer
  • You can be the Admin of the Application and delete any User or Advertisement
  • If you accidentally delete everybody from the Database, then please hit the Reset DB Menu Point, which brings back to Default state
